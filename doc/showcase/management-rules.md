## Règles de gestion : site vitrine A.I.R 

### Gestion des rôles

- **RG01** : Seuls les *conseillers d'administration* peuvent modifier le contenu du site vitrine de l'association.  
- **RG02** : Seul·e·s le directeur ou la directrice de l’association peuvent créer un compte de *conseiller d'administration*.

### Comptes et sécurité

- **RG03** : Un compte de *conseiller d’administration* doit obligatoirement contenir :
  - Un **nom** ;
  - Un **prénom** ;
  - Une **adresse mail unique** ;
  - Un **mot de passe**.

- **RG04** : Un **mot de passe** doit respecter les critères suivants :
  - Avoir une **longueur minimale de 8 caractères** ;
  - Avoir une **longueur maximale de 72 caractères** ;
  - Contenir **au moins un chiffre** ;
  - Contenir **au moins un caractère spécial**.

### Statistiques

- **RG05** : Le site vitrine doit comporter une section dédiée aux statistiques de l'association.
- **RG06** : Cette section doit afficher :
  - Le **nombre de centres d’hébergement**.
  - Le **nombre de personnes accompagnées** sur l’année précédente.
  - Le **nombre de logements attribués** sur l’année précédente.

### Parcours des publics

- **RG07** : Le site doit présenter un parcours dédié aux **demandeurs d'asile**.
- **RG08** : Le site doit présenter un parcours dédié aux **bénéficiaires de la protection internationale**.
- **RG09** : Chaque étape des parcours doit obligatoirement comporter :
  - Un **titre** ;
  - Une **courte description**.

### Offres d'emploi

- **RG10** : Le site vitrine doit comporter une section dédiée aux offres d’emploi de l’association.
- **RG11** : Chaque offre d’emploi doit obligatoirement comporter :
  - Un **titre** ;
  - Une **ville de localisation** ;
  - Un **type de contrat** ;
  - Une **courte description** ;
  - Une **image d’illustration**.
- **RG12** : Un bouton de **candidature** doit permettre aux visiteurs d’envoyer un mail à l’adresse de contact du siège.
- **RG13** : L’**objet du mail** généré doit automatiquement contenir le **titre de l’offre** concernée.

### Partenaires

- **RG14** : Le site vitrine doit comporter une section dédiée aux **partenaires** de l’association.
- **RG15** : Chaque partenaire doit obligatoirement comporter :
  - Un **nom** ;
  - Un **logo**.

### Historique

- **RG16** : Le site vitrine doit comporter une section dédiée à l’**historique** de l’association.
- **RG17** : Chaque événement de l’historique doit :
  - Renseigner une **année** ;
  - Contenir une **courte description**.

### Valeurs

- **RG18** : Le site vitrine doit présenter les **valeurs de l’association**.
- **RG19** : Chaque valeur doit :
  - Avoir un **nom** ;
  - Avoir une **image d’illustration**.

### Structures

- **RG20** : Le site vitrine doit présenter les **structures** de l'association.
- **RG21** : Chaque structure doit avoir une **description renseignée**.
- **RG22** : Aucune **information sensible** ne doit être publiée sur les centres d'hébergement.
- **RG23** : Seules les structures de type **CADA**, **CPH**, **HUDA** et **Centre de stabilisation** doivent voir leurs **missions précisées**.
- **RG24** : Seules les **adresses du siège**, du **centre FLE** et de la **boutique solidaire** doivent être affichées.
- **RG25** : Seuls le **siège** et la **boutique solidaire** doivent avoir un **numéro de téléphone** affiché.
- **RG26** : Les **horaires d’ouverture** de la boutique solidaire doivent être renseignés.
- **RG27** : Les **horaires des cours de FLE** doivent être renseignés.
- **RG28** : Un **lien vers la boutique solidaire** doit être accessible.

### Référencement

- **RG29** : Le site vitrine doit être **référencé sur les moteurs de recherche**.

### Gouvernance

- **RG30** : Le site vitrine doit présenter une section dédiée à la **gouvernance** de l’association.
- **RG31** : Cette présentation doit être **non-nominative**.

### Actualités

- **RG32** : Le site vitrine doit présenter les **dernières actualités** de l’association.
- **RG33** : Chaque actualité doit obligatoirement contenir :
  - Un **titre** ;
  - Une **courte description** ;
  - Une **image d’illustration**.
